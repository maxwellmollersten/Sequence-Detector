Sequence Detector

A hardware sequence detector implemented in VHDL on a Basys 3 FPGA, built as a project in the course Digital Electronics (1TE771, HT2025). The detector identifies the binary sequence 10110 from a stream of input bits driven by physical switches on the board. A LED lights up when the sequence is detected.

The design uses a finite state machine (FSM) with at least three flip-flops, fulfilling the course requirement of a minimum 5-bit sequence.

How it works

The target sequence

The FSM tracks progress toward detecting 10110 bit by bit. When the full sequence has been received in order, the output LED is asserted.

State machine

  • S0 — Idle, no bits matched
  • S1 — Matched 1
  • S2 — Matched 10
  • S3 — Matched 101
  • S4 — Matched 1011
  • S5 — Matched 10110 → LED on

Hardware

Components

  • Basys 3 FPGA board (xc7a35tcpg236-1)
  • Onboard switches — input bits and manual clock
  • Onboard LED — sequence detected indicator

Implementation

Written in VHDL and synthesized with Xilinx Vivado. Ports are mapped to physical pins using the Basys 3 constraint file.

The FSM is implemented as a Moore machine, where the output depends only on the current state.
